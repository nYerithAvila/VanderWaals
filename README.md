# Descripción breve del proyecto
En el presente proyecto encontrarás tableros interactivos donde resolvemos la pregunta central "Cuando los datos nos hablan sobre escasez hídrica, ¿Cuál es el impacto de nuestras acciones individuales y colectivas?". 

La idea central de nuestro proyecto consiste en acercar al usuario al avance del ODS6, específicamente en México, e identifique su capacidad de impacto ante la problemática de escasez de recursos hídricos.

Para esto hicimos uso de herramientas de código abierto como Python, Quarto, Git, Visual Code Studio. Además utilizamos base de datos abiertos como: CONAGUA, Agenda 2030, Aqueduct Water Risk Atlas.
